## Approach

```
1. Harness Engineering  ── 모델을 바꾸지 않고 시스템으로 AI 코딩 품질을 올린다
2. Verifiable AI        ── AI가 만든 코드를 자동으로 검증하는 파이프라인을 구축한다
3. Build First          ── 이론보다 돌아가는 시스템을 먼저 만들고, 거기서 배운다
```
