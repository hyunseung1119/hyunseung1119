## Approach

```
1. 하네스 엔지니어링 — 모델을 바꾸지 않고 시스템으로 품질을 올린다
2. 검증 가능한 AI — AI가 만든 코드를 자동으로 검증하는 파이프라인
3. 실무 중심 학습 — 이론보다 "돌아가는 시스템"을 먼저 만든다
```
